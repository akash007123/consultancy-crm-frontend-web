# Consultancy CRM

A comprehensive consultancy relationship management system built with React, TypeScript, Tailwind CSS, and shadcn-ui.

## Project Info

- **Frontend**: Vite + React + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + MongoDB
- **UI Components**: shadcn-ui

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- React Router DOM
- Zustand (State Management)
- React Query
- Recharts

## Getting Started

### Prerequisites

- Node.js 18+
- npm or bun

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev
```

The frontend will start on http://localhost:8080

### Build

```bash
# Build for production
npm run build
```
