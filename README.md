## 📌 Overview

A modern and scalable **Consultancy CRM Frontend** built using the latest React ecosystem.  
Designed for performance, clean UI, and developer productivity.

| Layer | Technology |

## 🚀 Getting Started

### ⚙️ Prerequisites

- Node.js 18+
- npm / bun

---

### 📦 Installation

```bash
# Install dependencies
npm install
▶️ Run Development Server
npm run dev

📍 App will run on:
👉 http://localhost:8080

🏗️ Build for Production
npm run build
📁 Project Structure
src/
├── components/     
│   ├── ui/          # shadcn-ui components
│   └── ...          # Custom reusable components
├── pages/           
│   ├── crm/         # CRM dashboard pages
│   └── ...          # Public pages
├── layouts/         # Layout wrappers
├── hooks/           # Custom React hooks
├── lib/             # Utility functions
├── store/           # Zustand state stores
└── data/            # Mock/static data
✨ Features
⚡ Lightning-fast UI with Vite
🎨 Modern design using Tailwind + shadcn-ui
🔐 Scalable architecture
🔄 API integration ready
📊 Dashboard & analytics support
📱 Fully responsive design
🔗 Backend Integration

Make sure backend is running:

👉 Default API base: http://localhost:3001

Update API base URL in your config if needed.

⚠️ Important Notes
Use environment variables for API URLs
Optimize builds before deployment
Keep components reusable & modular
⭐ Best Practices
Use React Query for API calls
Keep state minimal with Zustand
Follow component-based architecture
Use lazy loading for better performance
